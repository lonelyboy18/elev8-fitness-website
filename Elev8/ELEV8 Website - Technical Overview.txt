ELEV8 WEBSITE - TECHNICAL OVERVIEW
Last Updated: March 16, 2026

1) PROJECT SUMMARY
ELEV8 is a multi-page fitness website built with HTML, CSS, Bootstrap, JavaScript, PHP, and MySQL.
It supports a feedback workflow with server-side CRUD endpoints and frontend form validation.

2) TECHNOLOGY STACK
Frontend:
- HTML5 pages for content and navigation
- CSS3 custom styling in style.css
- Bootstrap 5 local assets in css/ and js/
- JavaScript validation and AJAX in validations.js

Backend:
- PHP endpoints in php/
- PDO-based database access (PHP Data Objects)
- MySQL schema in elev8_mysql.sql

Runtime:
- XAMPP (Apache + MySQL)

3) CURRENT ARCHITECTURE
Browser
  -> HTML pages + style.css + validations.js + Bootstrap assets
  -> AJAX requests to PHP endpoints
PHP Backend
  -> php/db_config.php for DB connection
  -> php/create_submission.php
  -> php/read_submissions.php
  -> php/update_submission.php
  -> php/delete_submission.php
Database
  -> MySQL database with submissions and users tables
